WebApplication Access and User Authentication

  1. User access the URL through browser using mobile;
  2. A landing page appears with two options for creation of profile or accessing existing account using “Register” or “Login” options respectively;
  3. The landing page has the option of Logging in the system using Email and Password;
  4. User can Register herself, by clicking Register button;
  5. An eight field form opens up to be filled by the user, Submit button will submit the form and register the user;
  6. Landing page also will have an option of “ Forgot Password”;
  7. Forgot Password page will receive new password and will update it after asking the confirmation modes.

View Profile

  1. Here, user can add Logo, Cover photo, Profile photo, Social media contact information, about of the company and other required fields;
  2. User can edit and update previous information provided as well.

View Dashboard

  1. The user can see how many views a card has received with reference to each month and other analysis related to it;
  2. User can edit and update previous information provided as well.

Change settings and View as

  1. Through this page , user can change password, update email and other account’s information;
  2. There is a URL of the card available here which can be shared and can be viewed as, allowing user to see how the Business Card will look like.
